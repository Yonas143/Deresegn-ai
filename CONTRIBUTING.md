# Contributing to ScanLogic

Thank you for your interest in contributing to ScanLogic! 🎉

## How to Contribute

### 1. Fork the Repository

Visit [github.com/yoniapp/Deresegn-ai](https://github.com/yoniapp/Deresegn-ai) and click "Fork"

### 2. Clone Your Fork

```bash
git clone https://github.com/YOUR-USERNAME/Deresegn-ai.git
cd Deresegn-ai
```

### 3. Create a Branch

```bash
git checkout -b feature/your-feature-name
```

### 4. Make Your Changes

- Follow the existing code style
- Add tests if applicable
- Update documentation

### 5. Test Locally

```bash
npm install
npm run dev
```

### 6. Commit and Push

```bash
git add .
git commit -m "Add: your feature description"
git push origin feature/your-feature-name
```

### 7. Create a Pull Request

Go to the original repository and click "New Pull Request"

## Areas We Need Help

- 🌍 **Translations** - Help translate the bot to more languages
- 🎨 **UI/UX** - Improve the landing page and user experience
- 🐛 **Bug Fixes** - Find and fix bugs
- 📚 **Documentation** - Improve guides and tutorials
- ✨ **Features** - Add new capabilities (see issues)

## Code Style

- Use TypeScript for type safety
- Follow existing patterns
- Add comments for complex logic
- Keep functions small and focused

## Reporting Issues

Found a bug? Have a feature request?

1. Check if it's already reported in [Issues](https://github.com/yoniapp/Deresegn-ai/issues)
2. If not, create a new issue with:
   - Clear description
   - Steps to reproduce (for bugs)
   - Expected vs actual behavior
   - Screenshots if applicable

## Questions?

Open a [Discussion](https://github.com/yoniapp/Deresegn-ai/discussions) or reach out to the maintainers.
